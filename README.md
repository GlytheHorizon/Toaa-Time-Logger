## 🕒 About the Project

> [!IMPORTANT]
> **Toaa Time Logger** is a high-performance web application designed to streamline time tracking and productivity analysis. It leverages modern cloud technologies and AI to provide a seamless user experience.
> 
> *Note: This project is built using Next.js 15 and Firebase, ensuring scalability and real-time responsiveness.*

---

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| **AI Integration** | Intelligent time analysis and suggestions powered by Google Genkit AI. |
| **Real-time Sync** | Instant data persistence across all devices via Firebase Firestore. |
| **Visual Analytics** | Beautiful, interactive charts and progress tracking built with Recharts. |
| **Premium UI** | Stunning dark/light mode interface using Radix UI and Tailwind CSS. |
| **Secure Auth** | Robust user authentication and data isolation through Firebase Auth. |
| **Responsive Design** | Fully optimized for mobile, tablet, and desktop viewing. |

## 🏗️ Developer Setup

### Prerequisites
- **Node.js** (v20+)
- **Firebase CLI** (`npm install -g firebase-tools`)

### Installation
```bash
# 1. Clone the repository
git clone https://github.com/GlytheHorizon/Toaa-Time-Logger.git

# 2. Enter directory
cd Toaa-Time-Logger

# 3. Install dependencies
npm install

# 4. Launch in Dev Mode
npm run dev
```
